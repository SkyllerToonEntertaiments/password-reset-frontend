Redefinir senha

Digite o e‑mail da conta e enviaremos um link para redefinir a senha.

Observação: o método resetPasswordForEmail() apenas envia o link. Após o click o usuário será redirecionado para seu app — então solicite a nova senha e chame supabase.auth.update({ password }).

Se você acessou este page via o link de recuperação, use o formulário abaixo para definir a nova senha.
